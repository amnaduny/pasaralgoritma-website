Belajar membuat website

Program ini dirancang untuk anak-anak dan remaja yang tertarik dengan pengembangan website. Mereka akan belajar dasar-dasar HTML, CSS, dan JavaScript untuk membangun tampilan dan fungsionalitas web. Selain itu, peserta juga akan memahami konsep desain responsif, interaktivitas menggunakan Canva serta debugging. Di akhir program, mereka akan membuat dan mempublikasikan website mereka sendiri.

Pengenalan HTML, CSS dan Javascript

Memberikan dasar pemahaman tentang HTML, CSS, dan JavaScript untuk mengenalkan konsep dasar pengembangan website, desain tampilan, serta interaktivitas web kepada anak-anak.

Pengenalan Desain dengan Canva

Mengajarkan anak-anak dasar-dasar desain grafis menggunakan Canva, termasuk pembuatan poster, presentasi, dan konten digital secara kreatif dengan elemen visual yang menarik.

Pengenalan Figma

Memberikan dasar pemahaman tentang Figma untuk mengenalkan konsep desain UI/UX, pembuatan wireframe, dan prototipe interaktif dalam pengembangan website dan aplikasi.
