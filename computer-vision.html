Apa yang akan dipelajari?

Dalam kursus ini, peserta akan mempelajari teknik dasar dan lanjutan dalam computer vision menggunakan Python dan pustaka populer seperti OpenCV, TensorFlow, Keras, dan PyTorch. Kursus ini bertujuan untuk membekali peserta dengan keterampilan untuk memproses dan menganalisis gambar dan video, serta membangun model deep learning untuk pengenalan objek, deteksi wajah, dan aplikasi lainnya.

Hasil Belajar

Setelah menyelesaikan kursus ini, peserta akan memahami :

  • Dasar-dasar Computer Vision: Memahami konsep dasar dalam computer vision, serta bagaimana memproses dan menganalisis gambar dan video menggunakan berbagai teknik.
  • Pengolahan Citra dan Deteksi Fitur: Menguasai teknik pengolahan citra dengan OpenCV, termasuk thresholding, deteksi tepi, dan deteksi serta pencocokan fitur.
  • Deep Learning untuk Computer Vision: Memahami dan menerapkan teknik deep learning, khususnya CNN, untuk tugas pengenalan citra dan deteksi objek.
  • Transfer Learning & Model Optimasi: Menerapkan transfer learning dengan model pre-trained untuk mempercepat pelatihan, serta melakukan tuning hyperparameter untuk meningkatkan kinerja model.
  • Penerapan Computer Vision pada Aplikasi Dunia Nyata: Membangun aplikasi real-time menggunakan computer vision untuk pengenalan wajah, deteksi objek, dan gesture recognition.
  • Segmentasi Citra & Optical Character Recognition (OCR): Menerapkan teknik segmentasi citra untuk analisis mendalam serta menggunakan OCR untuk membaca teks dalam gambar.
  • Deployment dan Evaluasi Model: Menerapkan model computer vision dalam aplikasi nyata dengan Flask, melakukan evaluasi performa, dan mengoptimalkan model untuk deployment yang efisien.

Modul ini akan memberikan pemahaman mendalam mengenai konsep-konsep dasar dalam Computer Vision, membantu Anda membangun fondasi yang kuat untuk pengembangan keterampilan analisis gambar dan video. Kursus ini mencakup berbagai teknik mulai dari pemrosesan gambar hingga penerapan model deep learning untuk computer vision, serta memberikan pemahaman yang diperlukan untuk bekerja dengan berbagai proyek computer vision nyata.

Computer Vision adalah bidang dalam kecerdasan buatan yang memungkinkan komputer untuk "melihat" dan memahami gambar atau video, mirip dengan cara manusia memproses visual.

OpenCV adalah pustaka yang digunakan untuk memproses dan menganalisis gambar. Hal ini mencakup operasi dasar seperti manipulasi gambar, konversi warna, dan transformasi geometris.
Thresholding digunakan untuk membagi gambar menjadi dua bagian (misalnya, objek vs latar belakang). Deteksi tepi menemukan batas objek dalam gambar, yang berguna untuk analisis lebih lanjut.
Color spaces (seperti RGB, HSV) digunakan untuk representasi warna dalam gambar. Image Filtering melibatkan penghapusan noise atau peningkatan fitur untuk analisis lebih lanjut.
Fitur seperti sudut atau garis dalam gambar dapat dideteksi dan dicocokkan untuk mencari kesamaan antara gambar atau objek dalam gambar.
Pembelajaran mendalam memungkinkan model untuk belajar dari data besar dan dapat digunakan untuk tugas computer vision yang kompleks seperti klasifikasi gambar dan deteksi objek.
CNN adalah jenis jaringan saraf yang sangat efektif dalam tugas computer vision, karena dapat mengekstraksi fitur otomatis dari gambar melalui lapisan konvolusi.
Proses melatih CNN untuk mengklasifikasikan gambar melibatkan pengumpulan data, pembuatan model, dan pengoptimalan parameter untuk menghasilkan akurasi yang tinggi.
Transfer learning menggunakan model yang telah dilatih sebelumnya pada dataset besar, kemudian disesuaikan untuk tugas tertentu, mengurangi waktu pelatihan dan meningkatkan akurasi.
Deteksi objek menggunakan CNN melibatkan menemukan lokasi dan jenis objek dalam gambar. Teknik seperti YOLO dan Faster R-CNN digunakan untuk tugas ini.
OpenCV memungkinkan pemrosesan video secara real-time, termasuk pembacaan video, analisis frame per frame, dan manipulasi seperti stabilisasi atau pengenalan objek.
Menggunakan teknik computer vision untuk mendeteksi dan mengenali wajah dalam gambar atau video. Ini digunakan dalam aplikasi seperti pengenalan wajah dan keamanan.
Mengenali gerakan tangan atau tubuh untuk interaksi dengan perangkat, sering digunakan dalam aplikasi kontrol berbasis gerakan dan perangkat augmented reality.
Segmentasi semantik memisahkan gambar ke dalam objek-objek yang memiliki makna, sementara segmentasi instance membedakan antara objek yang sama dalam gambar (misalnya, dua mobil yang berbeda).
OCR digunakan untuk mengubah teks dalam gambar menjadi teks yang dapat dibaca oleh mesin, sering digunakan untuk memindai dokumen dan membaca teks pada gambar.
Melacak objek yang bergerak di dalam video untuk mengikuti pergerakan atau menganalisis pola gerakan, digunakan dalam pengawasan dan analisis video.
Augmentasi gambar adalah teknik untuk memperbanyak data pelatihan dengan memanipulasi gambar (misalnya, rotasi, zoom, atau flipping) untuk meningkatkan generalisasi model.
Penyetelan hyperparameter melibatkan pengaturan parameter model untuk meningkatkan kinerjanya, sementara optimasi model berfokus pada pengurangan error dan peningkatan akurasi.
Membangun aplikasi yang memproses dan memberikan hasil computer vision dalam waktu nyata, seperti deteksi objek dan pengenalan wajah di aplikasi ponsel atau perangkat keras.
Edge computing melibatkan pemrosesan data dekat dengan sumber data (misalnya, perangkat atau sensor), memungkinkan pengolahan gambar atau video secara real-time dengan latensi rendah.
YOLOv5 adalah versi terbaru dari algoritma deteksi objek "You Only Look Once", yang dapat mendeteksi objek dengan akurasi tinggi dan kecepatan real-time.
Menggunakan computer vision untuk mendeteksi dan mengenali wajah secara real-time dalam video atau aplikasi streaming.
Flask digunakan untuk membuat aplikasi web yang mengintegrasikan model computer vision, memungkinkan pengguna untuk mengunggah gambar dan mendapatkan prediksi atau hasil deteksi objek.
Evaluasi model melibatkan pengukuran kinerja menggunakan metrik seperti akurasi, presisi, recall, dan F1-score. Setelah evaluasi, model dapat diterapkan dalam aplikasi nyata.
Proyek akhir melibatkan penerapan semua pengetahuan yang dipelajari untuk membangun aplikasi computer vision yang lengkap, seperti deteksi objek atau pengenalan wajah dalam video secara real-time.
