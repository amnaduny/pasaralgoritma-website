Apa itu Full Stack Web Development?

Full Stack Web Development adalah pengembangan aplikasi web yang melibatkan seluruh lapisan pengembangan, yaitu front-end (tampilan pengguna) dan back-end (logika dan server) serta pengelolaan database. Seorang Full Stack Developer mampu bekerja pada seluruh tumpukan teknologi aplikasi web, mulai dari antarmuka pengguna hingga pengolahan data di server dan penyimpanan di database.

Front-End Development

Front-end adalah bagian dari aplikasi web yang langsung berinteraksi dengan pengguna. Pengembang front-end bertanggung jawab untuk menciptakan pengalaman pengguna yang intuitif dan menarik melalui halaman web.

  • HTML (HyperText Markup Language): Struktur dasar dari halaman web.
  • CSS (Cascading Style Sheets): Desain dan tata letak halaman web untuk membuat tampilan yang responsif dan menarik.
  • JavaScript (JS): Bahasa pemrograman untuk menambahkan interaktivitas pada halaman web. Anda akan belajar cara manipulasi DOM (Document Object Model) untuk membuat elemen halaman dinamis.
  • React: Library JavaScript untuk membangun antarmuka pengguna yang dinamis dan efisien dengan komponen-komponen.

Back-End Development

Back-end adalah bagian dari aplikasi web yang berfungsi untuk mengelola data dan logika aplikasi, yang tidak terlihat oleh pengguna. Back-end mencakup server, API, dan database.

  • Node.js: Lingkungan runtime JavaScript yang memungkinkan pengembangan aplikasi sisi server. Node.js menggunakan JavaScript untuk membuat server dan API.
  • Express: Framework web minimalis untuk Node.js yang memungkinkan pembuatan server dan API yang efisien.
  • JWT (JSON Web Tokens): Digunakan untuk autentikasi dan otorisasi pengguna. JWT menyimpan data pengguna dan memungkinkan aplikasi memverifikasi identitas pengguna.
  • WebSockets (Socket.IO): Teknologi untuk komunikasi real-time antara client dan server, memungkinkan fitur seperti chat langsung dan pembaruan data secara real-time.

Database Management

Database digunakan untuk menyimpan, mengelola, dan mengakses data yang digunakan oleh aplikasi. Dalam kurikulum ini, Anda akan belajar cara menggunakan MongoDB, yang merupakan database NoSQL yang sangat populer.

  • MongoDB: Database NoSQL yang menyimpan data dalam bentuk dokumen (JSON). MongoDB sering digunakan dalam pengembangan aplikasi web karena fleksibilitas dan skalabilitasnya.

Deployment

Setelah aplikasi dikembangkan, langkah terakhir adalah deploy (menyebarkan) aplikasi ke server atau platform cloud agar dapat diakses oleh pengguna.

  • Heroku: Platform cloud untuk hosting aplikasi web. Heroku memungkinkan penyebaran aplikasi dengan mudah menggunakan Git, dan menyediakan berbagai fitur untuk mengelola server, seperti pengaturan lingkungan dan database.
